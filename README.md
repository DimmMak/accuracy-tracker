# 📊 Accuracy Tracker

> Performance Attribution Analyst for Blue Hill Capital.
> Converts stored hypotheses into measurable track record.

## What it does

Scores royal-rumble predictions vs real-world outcomes:
- Single-ticker hit rate
- Pair-relative hit rate (cleanest signal)
- Legend attribution (future)

## Commands

```
.accuracy                 → menu
.accuracy summary         → dashboard
.accuracy rumbles         → score all single-ticker calls
.accuracy pairs           → score all compare verdicts
.accuracy TICKER          → score one ticker's rumbles
.accuracy log [N]         → scoring runs history
```

## Install

```bash
./scripts/install.sh
```

## Why pair-relative matters

Single-ticker calls get confused with market beta. Pair-relative isolates skill from luck — this is how Millennium and Citadel grade their analysts.
